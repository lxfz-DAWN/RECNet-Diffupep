# RECNet-Diffupep
AI-peptide
## 项目概述

呼吸道病毒感染对公共卫生构成持续且严峻的威胁，然而基于天然序列的传统多肽药物设计方法难以主动开发出具有广谱抑制活性的候选药物。SARS-CoV-2等呼吸道包膜病毒拥有相似的细胞进入机制，均形成一个高度保守的6-HB结构，以此驱动病毒与宿主细胞的膜融合。
为此，本研究构建了一套人工智能驱动的多肽设计体系：我们开发了基于扩散模型的生成模型Diffupep，用于生成多肽序列；构建了基于蛋白质语言模型的结合预测网络RECNet，以评估多肽与靶点的相互作用；并完成了膜表达多肽高通量筛选实验体系的理论构建与预实验初步验证。结果表明，RECNet在多个标准数据集上表现出优良的预测性能，Diffupep亦已完成架构设计与预训练，展现出良好的条件生成能力。本研究建立了一套完整的方法学框架，在克服传统设计范式的局限性提供了可行路径与关键技术支撑。


## 系统架构

我们的一体化AI驱动多肽设计系统包含三个核心组件：

### 1. **Diffupep** - 条件多肽生成

- **模型**：基于扩散模型的序列生成器    
- **架构**：Diffuseq扩散模型，经过掩码训练和条件定向诱导训练    
- **能力**：实现多肽多种物理性质的定向进化    
- **创新**：拓展了非天然广谱多肽的设计空间    
### 2. **RECNet** - 蛋白-多肽相互作用预测
- **架构**：ESMC嵌入 → 多ESMC模块 → 多尺度深度卷积融合    
- **性能**：在多个金标准数据集上性能优于先前SOTA模型xCAPT5    
- **优势**：解决传统对接打分函数精度低、泛化差的问题
### 3. **实验验证框架**
- 膜表达多肽高通量筛选实验体系    
- 理论模型构建与预实验前期验证    
- 为打分网络和强化学习提供高质量数据库

## 安装指南

### 环境要求

- Python 3.8+    
- PyTorch 2.0+    
- CUDA 11.7+ 

### 快速开始

```bash
# 克隆仓库
git clone https://github.com/lxfz-DAWN/RECNet-Diffupep.git

# 安装依赖
pip install -r requirements.txt

```

## 项目结构

```bash
├── RECnetModel/              # 蛋白-多肽相互作用预测
│   ├── Use_Rosetta/         # Rosetta结构分析集成
│   ├── PanPep/              # 泛特异性多肽分析
│   ├── Use_Stripedhyena/    # 替代架构实验
│   └── The_last_chance/     # 最终模型实现
├── Diffupep/                # 扩散模型多肽生成
│   ├── diffuseq/            # 扩散序列建模核心
│   ├── generation_outputs/  # 生成的多肽序列
│   └── reward_model/        # 强化学习奖励模型
├── UPDmodel/                # 基于结构算法的模型
│   ├── Evolution/           # 进化算法
│   ├── ChaiRosettaScore/    # 基于Rosetta的打分
│   └── NetRobust/           # 模型鲁棒性测试
├── Dataset_washing/         # 数据预处理和清洗
└── Dataset/                 # 训练和评估数据集
```
